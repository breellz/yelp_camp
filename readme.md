## About The Project
A crowd-sourced review web-app for camps. It features yelp basic functionalities, user signup, camp review, comments and much more.

### Built With
* [NodeJs](http://nodejs.org)
  * Express
* [MongoDB](https://mongodb.com)
  * Mongoose
  * Passport-local
